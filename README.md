## Getting Started

Follow the instructures below to get setup for local development

1. Clone the repo

```sh
git clone https://github.com/vora/react-starter-kit.git
```

2. Start vite on port 3000

```sh
npm run dev
```

3. Run a production build

```sh
npm run build
```

4. Run prettier

```sh
npm run prettier
```

5. Run eslint

```sh
npm run eslint
```
